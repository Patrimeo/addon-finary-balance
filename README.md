Finary Balance addon for Patrimeo. Retrieves asset balances from Finary using sharing links and secure codes. Supports checking accounts, investments, fonds euro, commodities, real estates, and cryptocurrencies.

## Installation

```
composer require patrimeo/addon-finary-balance
```

## Usage

You need to share a Finary dashboard to make this addon work. Once it's done, Finary will provide you these two elements:

### Sharing Link
The sharing link is the link provided by Finary when you share a dashboard publicly.

### Secure Code
The secure code is the code that protects the sharing link, also provided by Finary.
